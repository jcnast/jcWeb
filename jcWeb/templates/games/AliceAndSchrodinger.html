Alice and Schrodinger

Alice and Schrodinger is a self-guided tour aimed at providing insight to the goings-on of the Quantum Nano Centre on campus at the University of Waterloo.Participants unlock information as they roam the building, learning about the building itself, the reasearch being done, and the future plans for the building.

The open house for which this application was built has completed, and so the tour-portion of the application can no longer be experienced. However, the data canbe manually unlocked in the settings page, so is still viewable by the user.

For more information and an analysis of the design/development process, please view the Alice and Schrodinger Post Mortem.

Controls

Everything is touch-based through UI components.

Supported Devices

This app is built for Android devices (2.3.3 and up) that have NFC and iBeacon capabilities. Unfortunately, we were unable to use iPhone's NFC chip readers.

Software Used
  • Game Engine: Unity 5
  • Languages: C#, Java, XML
