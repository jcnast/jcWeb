Sky Pirates

The islands hold valuable crystals that must be harvested for money. Unfortunately, they also keep the islands afloat! Co-operate with up to four players to fly around, collect the crystals and escape the collapsing island. The ship can have one pilot and three passengers while flying around. Be careful of the hostile island inhabitants and the turrets that will do massive damage to the ship and have protective shields around their gun barrels!

Unfortunately, we did not have time to create a solid interface for quitting the game, therefore: please run in windowed mode to more easily quit, and reference the supplied README file the controls.

For more information and an analysis of the design/development process, or if you would like to leave any comments about the game, please view the Sky Pirates Post Mortem.

Controls

A PC-compatible controller (Xbox, PS4, ...) is required to play this game. Up to 4 players can join at a time. Below are the controls with respect to each controller:

  • No character alive:
    • Y: Spawn character
  • As character:
    • Y: Interact (consume CRYSTAL when near, get into SHIP when near, get on SHIP if someone else is piloting ship)
    • A: Jump (HOLD to charge)
    • RT: Shoot
    • LS: Move
    • RS: Aim
  • As Ship:
    • Y: Get out/off
    • RS: Move
    • RT: Shoot

Supported Devices

This game is build for Windows computers, most computers should be able to run it.

Software Used
  • Game Engine: Unity 5
