The Escape of Sisyphus

Sisyphus is tired of endlessly pushing the boulder up the hill for eternity and has decided to escape! Help him avoid the guards sent to find him as he strives for freedom from never ending monotony. Get to the top portal to move on to the next level

As you go from level to level, more guards join the search to capture Sisyphus and force him back to work. As the number of guards increases, as does the number of hiding spots (tree stumps and boulders). The guards can only see a certain distance, and getting spotted or touching a guard will cause you to be captured, so be careful when you jump and get in.out of a hiding spot! Also, pay attention to how the guards move along their platforms - you may be able to discern some predicatable patterns and time your escapes accordingly!

For more information and an analysis of the design/development process, or if you would like to leave any comments about the game, please view the Escape of Sisyphus Post Mortem.

Controls

A keyboard is required to play this game, all controls are given in-game.

Supported Devices

This game is built for Windows computers, most should be able to run it. However, if you make it at higher levels it will likely start to run slower as the settings have not been optimized.

Software Used
