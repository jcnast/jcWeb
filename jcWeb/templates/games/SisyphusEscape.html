The Escape of Sisyphus

Sisyphus is tired of endlessly pushing the boulder up the hill for eternity and has decided to escape! Help him avoid the guards sent to find him as he strives for freedom from never ending monotony. Get to the top portal to move on to the next level, but be careful - as the level increases, so does the number of guards that have joined the search!

For more information and an analysis of the design/development process, please view the Escape of Sisyphus Post Mortem.

Controls

A keyboard is required to play this game, all controls are given in-game.

Supported Devices

This game is built for Windows computers, most should be able to run it. However, if you make it at higher levels it will likely start to run slower as the settings have not been optimized.

Software Used
  • Game Engine: Unity 4
  • Language: C#
  • Photo Editing: Gimp
