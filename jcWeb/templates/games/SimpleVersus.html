Simple Versus

Simple Versus is a 2D platform arena game where 2 to 4 players compete to be the last one standing. Damage your foes by outmaneuvering them on the platforms and off the walls while shooting them and trying to bounce off their heads. Players can use their ability to jump at varying heights and jump off of walls to avoid and out position their enemies.

This game was all hand made using C++ and SDL2 to render graphics and provide a way to handle various window events, such as joysticks and window events. I created this game in less than 4 weeks with no previous experience in SDL, and no previous experience in creating the workings of a game engine. I have plans to update the game, including multiple levels, additional character choices, improved sprites (as well as animations for the background and platforms), and several power-ups.

For more information and an analysis of the design/development process, or if you would like to leave any comments about the game, please view the Simple Versus Post Mortem.

Note: The game will spawn a character for the number of joysticks plugged in, with only characters that activate themselves during the main menu being active in the game loop. While characters handle their joysticks being unplugged and plugged in again after they are active, plugging in an originally unplugged controller will not create a new character in the menu, and so the game will need to be restarted to allow the new player to take part.

Controls

Game: Left joystick controls left/right movement, jumping (when on ground) and shooting (when in air) is any button

Menu: Any button to activate character, hold any button to start game (must have at least two characters active)

Win Screen: Press any button to return to menu, hold for rematch

Supported Devices

This game is made for Windows and Linux. Most versions should be supported, however please leave a comment on the Post Mortem if it does not work for you so I can try and resolve any issues.

Software Used
  • Game Engine: Hand Made using C++ and SDL2 for graphics
  • Languages: C++
