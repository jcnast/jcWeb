Games

Below are all the games I have had a part in making, including individual projects, games made at game jams, and professionally made games. Each game has a page that briefly describes the game and provides information about the software, libraries and languages used in the making of the game. In addition, there is a series of analysis, Post Mortem, that will dicuss in detail the process of making the game, the decisions that went into it, and reflecting on those decisions after completion. Additionally, if you would like to leave a comment about your opinion of the game or with some constructive criticism, you are able to do so on the Post Mortem analysis page for that respective game. Each game's page will also have a download link in the header bar that will either download the requried files to run the game, or link you to the appropriate download page.
